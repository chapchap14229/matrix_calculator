# Матричный калькулятор

Простое десктопное приложение для выполнения базовых операций над матрицами с графическим интерфейсом.

Этот проект — мой учебный полигон. В комментариях я подробно расписывал каждый шаг для себя, чтобы закрепить базу Tkinter

## Функциональность
- Поддержка матриц произвольного размера (выбор из выпадающего списка или ввод вручную)
- Операции с двумя матрицами:
  - Сложение (+)
  - Вычитание (-)
  - Умножение матриц (@)
- Операции с одной матрицей:
  - Определитель (determinant)
  - Транспонирование (transposition)
  - Обратная матрица (invertible)
- Удобный графический интерфейс на Tkinter
- Визуальное оформление (иконка приложения — Tomoko Math)

## Структура проекта
Всё реализовано в функциональном стиле в двух основных файлах:

- `matrix_gui_actual.py` — основной файл с интерфейсом (Tkinter), обработкой ввода, логикой вычисления,созданием полей матриц и вызовом функций расчёта
- `matrix_operations.py` — модуль с функциями математических операций над матрицами (сложение, умножение, определитель и т.д.) — **файл не используется в программе**
- `assets/` — папка для ресурсов (иконка `tomoko_math.png`)

Проект пока работает полностью в памяти (без базы данных), но структура готова к расширению (добавление сохранения истории, настроек и т.д.).

## Технологии
- **Python 3.14** — основной язык
- **Tkinter** — встроенная библиотека для GUI
- **NumPy** — для эффективных матричных операций
